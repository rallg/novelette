Full Interior Demo

For a demonstration, copy this entire novelette-demo folder to a place where you have file write privilege.

Compile demo.tex. On command line:

lualatex demo

That will produce the PDF, the usual TeX log file, and a special demo-summary.log file with interesting information (read it).

Subdocs

To see how subdocs work: Enter the various subfolders. Each *.tex file is a subdoc. Compile them one by one. Some of them have Preamble commands that show different results when compiled alone, or compiled in the main document. But the paragraph and page breaks are always the same.

Cover Image

To see how a cover image may be processed, you need program GraphicsMagic (a command-line-image processor), and the BASH command shell. It is available on Linux and Mac. If you are on Windows 10 or later, use the Linux subsystem for this. If you are on Windows <10, or do not have its Linux subsystem, then this procedure will not work.

Copy all of folder novelette-scripts to a writeable location. In the demo cover folder is image file riverboat.jp2. You may or may not be able to view it as-is, but this is unimportant. Copy this image file into novelette-scripts, then process:

bash cmyk4nvt riverboat.jp2

That takes awhile. It produces two files: riverboat-softproof.jpg and riverboat-nvtcN.jpg (where N is several digits). The softproof file shows what the image will look like, when printed to paper (more or less). You may discard it, if you wish. Move the riverboat-nvtcN.jpg file into the demo cover folder. Edit file demo-cover.tex so that \coverimage is the exact name of the new image file. Then compile:

lualatex demo-cover

That is very fast. The resulting PDF is PDF/X-1a:2001 format, with Output Intent "U.S. Web Coated (SWOP) v2", without attached icc color profile.

If you view the riverboat-nvtcN.jpg file in an image viewer, or view the PDF in a PDF reader, the colors you see are likely to be wrong. They may be much more colorful than you saw in the softproof. This is because most viewers and readers do not apply the correct color transform, when coverting the CMYK image (for printing) to RGB (for screen display).
