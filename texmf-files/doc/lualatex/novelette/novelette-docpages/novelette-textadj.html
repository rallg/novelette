Text Adjustment Settings

These are Preamble settings. They affect how text will be typeset, in main text and in some special locations. The purpose is to provide a uniform appearance throughout the book.

Name Style

Every chapter has a "name", which is its chapter title. The name may be just a number, or it may be a full line of text. It may not continue to more than one line.

Some non-chapter content may use the same name style. This might include Prologue, Epilogue, Introduction, About the Author, and so forth.

Command: \namestyle{align=A,font=F,scale=S}

See this example for how it is used.

  • align=A determines the horizontal alignment, where A is one of l c r o i (left, center, right, outside inside). Default align=c.
  • font=F determines which font will be used for the name text. F is one of main sans mono alt drama deco. Default font=main.
  • scale=S sets the scale of name text, relative to main font size. S is a number between 1 and 3 (decimals allowed). Default scale=1.6, which looks nice in chapter names.
  • This is a global setting. You may locally over-ride it.

^return

Subname Style

The use of subnames is optional. Typical use of a subname is beneath a chapter name, to provide a one-line description of the chapter. A subname may not extend to more than one line, but you may write several subnames.

Example: Chapter name is "3". Chapter subname is "At the Farmhouse".

Command: \subnamestyle{align=A,font=F,scale=S}

The settings have the same meaning as with \namestyle, except that the default subname scale is 1.25.

\setmicrotype{settings}

Novelette uses the microtype package for improved appearance of body text. Since the package does not load until after Preamble, this command provides a way to "PassOptionsToPackage" when microtype loads.

If you do not know what that means, simply ignore it. Novelette is pre-set to values that are good for its purpose. You do not need this command.

If you do wish to play with this command, Novelette will not let you use the microtype draft mode. Microtype will always be in its own final mode, no matter what you choose via \mode in Novelette.

^return

\myparindent{size}

Novelette pre-sets the default paragraph indent at 1.5em. That looks good for the purpose intended. But if you need a different default value, use \myparindent.

  • Example: \myparindent{10pt} for a small indent.
  • This is a global setting. If you need to locally change the indent, that is performed using other commands in body text.

^return

\warnpercent{on or off}

TeX uses the percent character % for comments. Anything following %, on the same line of raw text (prior to typesetting) will be ignored.

If you need to print the percent character, you may use a backslash \% there.

Default \warnpercent{off} does nothing. But \warnpercent{on} issues a Warning if you use the un-escaped % anywhere in body text. It does not warn for Preamble. This is a way to detect accidental use of % (which means comment) when you really meant to print %.

In body text, you may use \memo{This does not print.} where you need it. The difference between memo and %, is that after the memo completes, anything following it will print.

If you use this to issue a Warning, then the PDF will not be in final mode.

The novelette-docpages directory has a file named "warnmebro.tex". You may edit it with warnpercent on or off, for comparison.

^return

\warnquotes{on or off}

TeX normally converts straight double quotes " to right curly quotes ‚Äù. Alas, some writers (moi) are in the habit of typing " with the expectation that a smart-quotes feature will alternately convert them to left and right curly quotes, as needed. That is not what TeX does, so the print will be wrong. This is often hard to notice.

Default is \warnquotes{off}. If you use \warnquotes{on} then Novelette will issue a Warning if you wrote " anywhere in the document body. It does not warn for Preamble.

Since Novelette uses UTF-8 text encoding, you may directly paste left and right curly quotes from a character map. Or, you may use `` (two acute characters, also known as backticks) for left curly quotes, and two '' (straight single quotes, also known as straight apostrophe) for right curly quotes.

If you use this to issue a Warning, then the PDF will not be in final mode.

This documentation directory has a file named "warnmebro.tex". You may edit it with warnquotes on or off, for comparison.

^return

\warnalert{on or off}

Novelette sometimes writes ALERT messages to the log file. These are not Errors, or Warnings. Usually, an ALERT means that you set something to an unusual value. Did you really mean that, or not?

Default \warnalert{off}. If you use \warnalert{on} then a Warning will be issued if there were any Novelette-caused ALERTs.

It may be that the ALERT was thrown by a situation that you should fix. Or maybe it is what you intended, so no need to fix it. When you are satisfied that nothing needs to be fixed, turn \warnalert{off}.

If you use this to issue a Warning, then the PDF will not be in final mode.

This documentation directory has a file named "warnmebro.tex". You may edit it with warnalert on or off, for comparison.
