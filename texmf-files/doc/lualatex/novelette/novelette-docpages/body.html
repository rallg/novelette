Terminology

The document body is everything between (but not including) \begin{document} and \end{document}. This is where you write your book.

If you write anything following \end{document}, it will be ignored. So, if you need to write a note to yourself, you can put it there.

Code Example
\frontmatter ... \mainmatter ... \backmatter

% Preamble is up here.
\begin{document}
\frontmatter
Content of frontmatter
\mainmatter
Content of mainmatter.
\backmatter % Rarely used.
Content of backmatter, if any.
\end{document}
Anything written here will be ignored.

Using Subdocs

Best practice is to divide your book into a number of component files, each of which can be edited independently. Then you can compile just one portion at a time, as you write and edit.

Although LaTeX has several ways to do that, Novelette does it one way: using subdocs. You will be so happy! The use of subdocs is described on another page.

What is Frontmatter?

Front Matter, or frontmatter is a standard publishing term. It refers to the material that precedes the start of your story.

Novelette expects that the frontmatter will be exactly six pages, with very specific construction. This is explained in detail on another page.

What is Main Matter?

Main Matter, or mainmatter, is your story. It includes all chapters, and possibly things such as Prologue, Epilogue, and anything else that is part of the story. If there is a fictional Introduction (perhaps pretending to be written by an archaeologist, who discovered the rest of the story in a lost manuscript), then it is also mainmatter.

Mainmatter re-sets printable page numbering to arabic 1. If the layout has header or footer, it will print, except header will not print above chapter openings. Note: Internal PDF page numbering is not re-set.

If possible, write your story so that it ends mid-way on a recto page. This presents the most pleasing appearance, especially if you have something in backmatter.

What is Back Matter?

Back Matter, or backmatter is material added after the story ends. It is common in academic non-fiction, where there may be a bibliography, endnotes, appendix, index. But in the case of popular fiction, it is very rare to use backmatter.

Sometimes, you may place a very short "About the Author" in backmatter. If you have Acknowledgements, you may put them here. Commercially published books may show movie posters, advertisements (no joke!) or other irrelevant material. Don't do that.

If you use backmatter, it does not reset the page numbering, but no page numbers or header/footer will print.

The Final Page

No matter how your book ends (with or without backmatter), Novelette will automatically add one or two blank pages, so that the final page of the PDF is blank verso. This is required by print-on-demand. In some cases, the manufacturer will add more blank pages.

The final verso page (blank in your PDF) may have a manufacturer code added, at time of printing. This is not within your control.
