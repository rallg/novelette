Introduction

LaTeX has a number of built-in commands for changing text styles: different sizes, different weights, and so forth. But most of them do not work in Novelette. No \small, no \large, no \textbf, and no similar commands.

This is intentional. Most built-in LaTeX styles are incompatible with the uniform line grid and text density that is expected in popular fiction. So, Novelette re-defines the known LaTeX commands: either they are silently ignored (text appears normal), or an error or alert is thrown, depending on the command.

For best results, use only Novelette's own commands for styling text. They are shown on this page.

Page numbers and running headers are styled using \layout{setting}, described on another page.

\namestyle{setting} % Preamble only.

The \namestyle globally sets how text will appear in any \name{text} command. Then, each instance of \name presents a uniform appearance.

  • Setting: \namestyle{align=A,scale=S,font=F}
  • align=A : A is one of l r c o i. Horizontal alignment left, right, center, outside, inside. Default c
  • scale=S : S is a number >= 1, which scales the text size, relative to main text size. Default 1.6.
  • font=F : F is one of main book light med dark. Chooses which font will be used for the text. Default main.

\name{text} % In document body.

The \name command prints its text according to the style set by \namestyle in Preamble. This is most useful for items such as chapter title.

You may use this command more than once. It may only be used within these environments: upperpage fullpage contentspage.

  • font=F : F is one of main book light med dark. Chooses which font will be used for the text. Default main.

\subname{text} % In document body.

The \subname command prints its text according to the style set by \subnamestyle in Preamble. This is most useful for items such as chapter subtitle.

You may use this command more than once. It may only be used within these environments: upperpage fullpage contentspage.

The \name and \subname commands may be used in any order.

\formatXstyle{text} % Preamble only.
(where X is one of: a b c d e f)

Settings \formatastyle through formatfstyle globally set how text will appear in the corresponding \formata{text} through \formatf{text} commands.

  • Setting: \formatastyle{align=A,scale=S,font=F}
  • align=A : A is one of l r c o i. Horizontal alignment left, right, center, outside, inside. Default c
  • scale=S : S is a number >= 1, which scales the text size, relative to main text size. Default 1.
  • font=F : F is one of main book light med dark. Chooses which font will be used for the text. Default main.
  • Likewise for \formatbstyle \formatcstyle \formatdstyle \formatestyle \formatfstyle.

\formatX{text} % In document body.
(where X is one of: a b c d e f)

The \formatX command prints its text according to the style set by \formatXstyle in Preamble, where X is one of a b c d e f.

You may use these commands more than once. It may only be used within these environments: fullpage contentspage.

\smalltext{text} % In document body.

This command prints the text at slightly smaller than normal size, with slightly closer line spacing. It may include more than one paragraph.

It is useful when you need to provide a description of a chapter, beneath the chapter title, but before the chapter's main text begins. It is also useful if you have a lot to say on the copyright page.

  • You may use \smcp \allsmcp \onum \lnum \ital within \smalltext, but not other style commands such as \name.
  • You may use \smalltext only within an upperpage, fullpage, or copyrightpage.

\scenestyle{setting} % Preamble only.

The \scenestyle globally sets how text will appear in any \scene{text} command. Then, each instance of \scene presents a uniform appearance. Does not affect \scene{} or \scene{-} or \scene{*}.

  • Setting: \scenestyle{align=A,scale=S}
  • align=A : A is one of l r c o i. Horizontal alignment left, right, center, outside, inside. Default c
  • scale=S : S is a number >= 1, and <=1.25, which scales the text size, relative to main text size. Default 1.14. Notice that the maximum value is 1.25.
  • Unlike most other style commands, you cannot choose a font. Always main. If you write a font setting, it will be ignored.
