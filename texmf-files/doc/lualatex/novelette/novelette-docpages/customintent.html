Custom Intent

If you produce PDF/X format, it requires an Output Intent. If the intent is not one of the three built-in choices (SWOP, FOGRA39, JC200103) then you can provides another. Note that the FOGRA39 choice is actually FOGRA39L.

Whatever intent you provide, it will almost certainly be ignored by the printer. So, do not waste your time doing this, unless you must.

In Preamble:

\intent{custom}
\customOIidentifier{string}
\customOIcondition{string}
\customOIinfo{string}
\customOIregistry{string} % Optional.

Notes:

  • You must set the identifier, condition, and info.
  • The default registry is http://www.color.org if you do not set it.
  • You can find industry standard string enries at the above registry.
  • Within each string you must precede certain special characters with backslash: \% \_ \$ \& \@ \~ \#
  • If you do not set \intent{custom}, then the customOI settings will be silently ignored.

  • Important: Whether you are creating the book interior or color cover, Do NOT write an entry for color profile (*.icc file).
