Custom Intent

If you produce PDF/X format, it requires an Output Intent. If the intent is not one of the three built-in choices (SWOP, FOGRA39, JC200103) then you can provides another. Note that the FOGRA39 choice is actually FOGRA39L.

Whatever intent you provide, it will almost certainly be ignored by the printer. So, do not waste your time doing this, unless you must.

In Preamble: \intent{custom} then write settings for each of the following:

  • \OIidentifier{string}
  • \OIcondition{string}
  • \OIinfo{string}
  • \OIregistry{string}

The various string entries must comply with industry standards. You can find them at http://www.color.org, which is also the \OIregistry for nearly all intents.

You may look at file "novelette-metadata.sty" in a plain text editor, to see how it is done for the three built-in intents. This file is in texmf-dist/tex/lualatex/novelette.

Be careful: Some characters, such as % _ # & have special meaining in TeX. If they appear in any string they must be excaped by a backslash, such as \%.

DO NOT write an entry for color profile (*.icc file).
