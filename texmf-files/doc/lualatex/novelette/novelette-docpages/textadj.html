Text Adjustment Settings

These are Preamble settings. They affect how text will be typeset, in main text and in some special locations. The purpose is to provide a uniform appearance throughout the book.

Name Style

Every chapter has a "name", which is its chapter title. The name may be just a number, or it may be a full line of text. It may not wrap to more than one line.

The same name style may be useful in other places, such as Prologue, Epilogue, Introduction, and so forth. The decision is yours.

Command: \namestyle{align=A,font=F,scale=S,onum|lnum,ital}

See openings for how it is used.

  • align=A determines the horizontal alignment, where A is one of l c r o i (left, center, right, outside inside). Default align=c.
  • font=F determines which font will be used for the name text. F is one of main sans mono alt drama deco. Default font=main.
  • scale=S sets the scale of name text, relative to main font size. S is a number between 1 and 3 (decimals allowed). Default scale=1.6, which looks nice in chapter names.
  • onum|lnum forces the style of numerals in \name. By default, numerals are the same style as main text. Use onum to force OldStyle, or lnum to force Lining.
  • ital writes the \name in Italics. In many cases, it is better to locally use \ital to style only some characters.
  • This is a global setting, in Preamble. You may locally over-ride it as optional argument to any \name command.

Subname Style

The use of subnames is optional. Typical use of a subname is beneath a chapter name, to provide a one-line description of the chapter. A subname may not wrap to more than one line, but you may write several subnames.

Example: Chapter name is "3". Chapter subname is "At the Farmhouse".

Command: \subnamestyle{same choices as namestyle}

The default subname scale is 1.25.

\setmicrotype{settings}

Novelette uses the microtype package for improved appearance of body text. Since the package does not load until after Preamble, this command provides a way to "PassOptionsToPackage" when microtype loads.

If you do not know what that means, simply ignore it. Novelette is pre-set to values that are good for its purpose. You do not need this command.

If you do wish to play with this command, Novelette will not let you use the microtype draft mode. Microtype will always be in its own final mode, no matter what you choose via \mode in Novelette.

\myparindent{size}

Novelette pre-sets the default paragraph indent at 1.5em. That looks good for the purpose intended. But if you need a different default value, use \myparindent.

  • Example: \myparindent{10pt} for a small indent.
  • This is a global setting. If you need to locally change the indent, that is performed using other commands in body text.
