Page Layout

This page does not discuss cover artwork.

Command: \layout{style=S,hgap=H,fgap=F,num=N}

  • style=S requires S=integer, 1 to 7. Each number selects the global page style: Whether it has a header and/or footer, and how items are aligned there. The choices are described in detail, below.
  • hgap=H is the header gap. It is the additional line space from the header to the top line of main text. The value of H is a decimal number from 0 to 1. Value 0 is rarely used, because it would place the header immediately above main text, with no extra space. Value 1 produces a whole empty line between header and main text. A value of 0.5 is often used. If there is no header isn the page style, the hgap setting is ignored.
  • fgap=F is the footer gap. It is the additional line space between the last line of main text and the footer. Same considerations as with hgap.
  • num=N set the style of numerals used for page numbers. It is a one or two letter code, described below.

When layout uses a header, then every page reserves space for a header, even if nothing is printed there. Same with footer.

When the numeral style uses Italics, then lowercase roman numerals (sometimes seen in frontmatter) will also be italicized. Note that the numeral style does NOT determine whether roman or arabic numerals are used; that is determined by frontmatter and mainmatter.
