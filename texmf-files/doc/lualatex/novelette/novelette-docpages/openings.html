Openings and Displays

This page is under development.

 

Each chapter begins with an opening, which visually indicates that something new will follow. Some non-chapters may also use an opening. Within the opening, styled text shows the chapter name, and optional subname. After the opening, main text begins lower on the same page.

A display page stands by itself. No main text follows. Half-title, title, and copyright pages are all displays. Other possible displays are dedication, epigraph, and some others. On a display page, you may style text, and position text and images, with more freedom.

The Opening Environment

Each new chapter, and some things that look like a chapter, start with an opening. This is not invoked automatically. You must do it yourself. That is, Novelette does NOT have a \chapter command.

In the above image:

  • The opening is the area above start of main text. It is created by the opening environment: \begin{opening} through \end{opening}
  • No header appears above an opening. If the layout has a header, its space is reserved, but empty.
  • If the layout has a footer, it will appear, unless you write \thispagestyle{empty} within the opening.
  • Within the opening, you may place a name, using \name[options]{text of name}. The default style is set using \namestyle{setting} in Preamble. If you wish, the default style may be locally changed using options in the same command format as \namestyle.
  • You may also add a \subname[options]{text of subname}.
  • The name and (if used) subname cannot wrap. If you need a name or subname with more than one line, you may repeat the commands.
  • The opening may contain an image, using ordinary \image.
  • You may write normal-size text (but not main text) in the opening, perhaps if you wish to provide a short summary of what is to follow. In this case, the \blockindent environment may be useful
  • The first line of main text, immediately following \end{opening, is not indented. This is standard style.
  • The \init command may be used to style the first letter, and first few words of main text. This is very attractive.
  • Vertical positioning of name and subname, and main text, is not automated. You must manually add blank lines, where needed, using \null for each blank line.
  • This particular book has scenes, using the \scene command. Most books do not use scenes.

Here is the code for the above image. The styles used for name and subname were set in Preamble:

\begin{opening}
\null\null\null\null\null\null
\name{3. The Mississippi River}
\null
\subname{Excerpt from \ital{Life on the Mississippi}, 1883}
\null\null\null\null\null
\end{opening}
 
\init[-.1]{T}{he Mississippi} is well worth reading about. It is not a commonplace river, but on the contrary is in all ways remarkable.
 
\scene{+I. As a River}
 
Considering the Missouri its main branch, etc.

Display Pages

A display page stands by itself. Most or all pages in frontmatter are display pages. A blank page is a display page, but it does not use the display environment: \blankpage is all.

Most of the time, mainmatter has no display pages. If your story is divided into parts, then a disaply page may be used as separator. If you add "About the Author" or something similar after then end of the story, it might be a display page.

If a full-page image is within main text, it is NOT a display page. Instead, it uses the ordinary image command.

\begin{display}
\style[options]{Some Text}
\style[options]{More Text}
\image[options]{filename.png}
\style[options]{and so forth}
\end{display}

The display environment starts a new page. If the layout has a header, it will be empty. When the display ends, whatever follows will be on another new page.

The \style command allows you to change the font, size, and alignment. Within the style, text may wrap to more than one line.

On a display page, lines are usually off-grid. This is not a problem, because main text is not used there.

You may also use images.

The \style command may only be used within a display.
