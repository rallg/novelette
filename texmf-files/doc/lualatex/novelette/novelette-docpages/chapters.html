What Is a Chapter?

This page uses the word "chapter" to mean anything that is styled like a chapter, with a chapter title (and possibly other material) in the upper part of the first (or only) page, and text beginning lower on that page.

So, "chapter" might include things such as a short "Author's Note" in frontmatter, if it has the appearance of a chapter. It might include a one-page "About the Author" in backmatter.

Chapter Start

\begin{upperpage} ... \end{upperpage}

Each chapter begins with an upperpage environment. This starts a new page. Within this environment, you may use \name and \subname to style text for the chapter title/subtitle. You may place an \image. You may provide a short chapter description using blockindent and \smalltext.

Within upperpage, text may be off-grid, due to the varying sizes. But when upperpage ends, Novelette calculates and adds some line space, so that the following main text is on-grid.

No header prints above upperpage. If there is a footer, it will print in mainmatter. This behavior may be locally changed using \thispagestyle{setting}.

Novelete has no provision for the style of writing that quickly moves from one short chapter to another, mid-page.

Main Text

Main text will begin on-grid, but at no particular vertical position, because the amount of content within upperpage may vary.

For uniform vertical position in each chapter, place guides at the desired locations. Use \null to insert a blank line within upperpage. But do not use \null between the end of upperpage and the start of main text.

The first line of main text will not be indented. This is standard. You may style the very beginning of main text using \init, as described below.

Initial Main Text

\init{letter}[text]

In almost all books (fiction or not), the start of a chapter's main text has special styling. Nowadays, the "drop cap" is rare, and Novelette does not allow it. The "raised cap", often followed by some small caps text, is very common:

\init{D}[uring the day,] they slept. When evening came...etc.

\init{``¿Q}[ue pasa?''] ella me preguntó.

Command \init{letter}[text] does these things:

  • A blank line is inserted above, so that the letter does not bump into anything in upperpage.
  • The letter is scaled to 1.7x its normal size. If it has a descender (JQÇ¡¿) then Novelette automatically adjusts the descender, so that it does not bump into the following line of text.
  • Normally, the letter is just one uppercase letter. However, if the line begins with punctuation (quotes, guillemet, apostrophe, inverted exclamation or question) then include the punctuation with the letter. It will be automatically adjusted.
  • The optional text will be printed in small caps (only lowercase). Do not attempt to include the entire first line of text. Note that text does not duplicate the initial letter:
    \init{D}[uring the day], they slept.
  • If letter is the letter T, (or punctuation followed by T), then kerning is applied automatically.
  • You may not use \icon within \init. But you may place an icon before or after the complete \init
  • When used as first line of main text, following the uppercase environment, or when used as the first line of a scene, following the \scene command, the line is automatically not indented. If you prefer indentation there, use \forceindent first.

\scene{text}

If a chapter has scenes, then you must separate them, using some kind of visual cue. The \scene command serves this purpose.

  • \scene{} or \scene: Empty two-line gap.
  • \scene{*} or \scene* : Two-line gap. Positioned midway in the gap are three asterisks, centered and widely separated.
  • \scene{-} or \scene- : Two-line gap. Positioned midway in the gap is a centered, short horizontal line.
  • \scene{text} (text is anything else) : Gap of three lines. Slightly below the middle line (off-grid), the text is printed according to the \scenestyle{setting} in Preamble, or its default setting.

Scenes are not automatically numbered. Do it yourself!

After \scene, the first line of following text is not indented. You may use \init there, but this may be visually cluttered.

If \scene happens to occur near the bottom or top of a page, or if its gap would split pages, then Novelette will attempt to automatically compensate (if possible). But in many cases this is not possible, so you will need to edit the text. An alert message will appear in the summary log.
