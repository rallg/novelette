Terminology

This page uses the word "chapter" to mean anything that is styled like a chapter, with a chapter title (and possibly other material) in the upper part of the first (or only) page, and main text beginning lower on that page.

So, "chapter" might include things such as a short Author's Note in frontmatter, if it has the appearance of a chapter.

Chapter Style Settings in Preamble
\namestyle{setting} \subnamestyle{setting} \scenestyle{setting}

Commands \name{text}, \subname{text}, and \scene{text} are described below.

In order to present a uniform appearance wherever those commands are used, you may set their styles in Preamble. For use with \name :

  • \namestyle{align=A,scale=S,font=F}
  • align=A : A may be one of l r c o i. Horizontal alignment left, right, center, outside, inside. Default c
  • scale=S : S is a number >= 1, which scales the text size, relative to main text size.
  • font=F : F may be one of main light dark. Chooses which font will be used for the text.
    • Default font is main, which is Swainson (same as main font). This allows Regular, Italics, small caps.
    • Font light is a "titling" style, Regular only. Its lowercase letters are reduced-size uppercase, so it does not have separate small caps.
    • Font dark provides a dramatic effect. All of its letters are Regular uppercase (if you type lowercase, you see uppercase).

The same considerations apply to \subnamestyle and \scenestyle. Default scale is 1.6 for \namestyle, 1.14 for \subnamestyle and \scenestyle.

The maximum scale for \scenestyle is 1.25. Although you can change the font, it is usually better to keep it as default main so that scene changes are not too prominent.

These styles are global. You cannot change them, once set. You may set all or part of the values. Example: \namestyle{scale=1.8,font=light} sets the scale and font, and keeps the default center alignment.

The Upperpage Environment
\begin{upperpage}...\end{upperpage}

Each new chapter begins with an upperpage environment. This starts a new page, which allows distinctive layout in its upper portion. After upperpage ends, ordinary text follows on the same page. This text will be on-grid, even if the material in upperpage is off-grid.

The height reserved for the upperpage, and the content within it, are not automatically generated. That is your responsibility. In shade mode, the guides can help you align material, for uniform heights.

Within upperpage you may use \name, \subname, \image, \icon, and \smalltext. You may also use the \formatX commands, where X=a,b,c,d,e,f. Although \name suggests chapter title, and \subname suggests chapter subtitle, and \smalltext suggests chapter summary, none of these commands has a particular meaning or purpose. Use them, or not, in any order, possibly multiple times.

If you write a \footnote within upperpage, no marker is placed, either there or at the footnote itself. This style is used when the footnote is a general comment, rather than specific.

You cannot use upperpage inside the copyrightpage or dedication environments. You cannot use upperpage in the middle of a chapter (because it would break the chapter into two pieces).

If page layout uses a header, it will be empty above upperpage. If page layout uses a footer, it will be empty in frontmatter, but will show the page number in mainmatter.

After upperpage ends, the first line of main text is not intended. You may use the \init command to apply styling there.

Here is a typical upperpage. Its code is shown lower on this page:

\name{text}

The \name command prints its text according to the style set by \namestyle in Preamble. This is most useful for items such as chapter title.

You may use this command more than once. Each instance is independent.

\subname{text}

The \subname command prints its text according to the style set by \subnamestyle in Preamble. This is most useful for items such as chapter subtitle.

You may use the command more than once. Each instance is independent.

The \name and \subname commands may be used in any order.

\init{letter}[text]

In almost all books (fiction or not), the start of a chapter's main text has special styling. Nowadays, the "drop cap" is rare, and Novelette does not allow it. The "raised cap", often followed by some small caps text, is very common:

\init{D}[uring the day,] they slept. When evening came...etc.

\init{``¿Q}[ue pasa?''] ella me preguntó.

Command \init{letter}[text] does these things:

  • A blank line is inserted, so that the letter does not bump into anything above it.
  • The letter is scaled to 1.7x its normal size. If it has a descender (JQÇ¡¿) then Novelette automatically adjusts the descender, so that it does not bump into the following line of text.
  • Normally, the letter is just one uppercase letter. However, if the line begins with punctuation (quotes, guillemet, apostrophe, inverted exclamation or question) then include the punctuation with the letter. It will be automatically adjusted.
  • The optional text will be printed in small caps (only lowercase). Do not attempt to include the entire first line of text. Note that text does not duplicate the initial letter:
    \init{D}[uring the day], they slept.
  • If letter is the letter T, (or punctuation followed by T), then kerning is applied automatically.
  • When used as first line of main text, following the uppercase environment, or when used as the first line of a scene, following the \scene command, the line is automatically not indented. If you prefer indentation there, use \forceindent first. If you use \init in some other location (unusual), it will indent; you may use \noindent or \backindent if desired.

\scene{text}

If a chapter has scenes, then you must separate them, using some kind of visual cue. The \scene command serves this purpose.

  • \scene{} or \scene: Empty two-line gap.
  • \scene{*} or \scene* : Two-line gap. Positioned midway in the gap are three asterisks, centered and widely separated.

  • \scene{-} or \scene- : Two-line gap. Positioned midway in the gap is a centered, short horizontal line.
  • \scene{text} (text is anything else) : Gap of three lines. Slightly below the middle line (off-grid), the text is printed according to the \scenestyle{setting} in Preamble, or its default setting.

Scenes are not automatically numbered. Do it yourself!

After \scene, the first line of following text is not indented. You may use \init there, but this may be visually cluttered.

If \scene happens to occur near the bottom or top of a page, or if its gap would split pages, then Novelette will attempt to automatically compensate (if possible). But in many cases this is not possible, so you will need to edit the text. An alert message will appear in the summary log.
