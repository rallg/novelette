Chapters

This page is under development.

 

Each new chapter begins with an upperpage environment. This starts a new page, for distinctive layout in its upper portion. You may also use an upperpage for non-chapter materials: Prologue, Epilogue, and perhaps others such as Introduction, Author's Note, Acknowledgements, etc.

The height reserved for the upperpage, and the content within it, are not automatically generated. Commands \name and \subname provide a uniform style. In shade mode, the \guides help with vertical alingment. The \init command may be used for the first letter and first few words of the main text.

Here is a typical upperpage. Its code is shown at bottom of this page:

The upperpage Environment

\begin{upperpage} content \end{upperpage}

An upperpage always starts a new page, so it is always at the top of its page. No header appears above an upperpage, but if the layout scheme uses a header, its empty space is reserved there.

If the layout scheme uses a footer, it will be empty in frontmatter, and will display its usual content (page number) in mainmatter. You may change this per-page using \thispagestyle{footer} and \thispagestyle{empty}.

If you write a \footnote within the upperpage, it does not place a reference marker. The text of the footnote will appear at the bottom, as usual, but withut a referenc marker. This is standard style. It is used when the note pertains to the whole chapter, instead of particular sentence.

Within the upperpage, you may use \name and \subname to print styled text. You may also use images and icons.

Some authors (this is rare) like to provide a brief summary of the chapter contents. If you do this, be sure that it can be visually distinguished from main text. Usually, \blockindent or \centering is appropriate.

Main text begins immediately after \end{upperpage}. Its first line is not indented. You may use the \init command to style an initial amount of main text.

To create a blank line at the beginning, end, or between items in an upperpage, use \null.

In shade mode, \guides help you to visually align items, so that they appear the same for each chapter.

\name[option]{one line of text}

The \name command prints its text according to the style set by \namestyle in Preamble. This is most useful for items such as chapter title.

You may locally change the horizontal alignment (nothing else) using option [align=A], where A is one of l c r o i for left, center, right, outside, inside.

You may not break or wrap lines, or add a paragraph, within \name. However, you may use the command more than once, for multi-line material. Each instance is independent.

\subname[option]{one line of text}

The \subname command prints its text according to the style set by \subnamestyle in Preamble. This is most useful for items such as chapter subtitle. Not every chapter needs a subname.

You may locally change the horizontal alignment (nothing else) using option [align=A], where A is one of l c r o i for left, center, right, outside, inside.

You may not break or wrap lines, or add a paragraph, within \subname. However, you may use the command more than once, for multi-line material. Each instance is independent.

The \name and \subname commands may be used in any order.

Raised Capital at Start of Main Text

In almost all books (fiction or not), the start of main text has special styling. Nowadays, the "drop cap" is rare, and Novelette does not allow it. The "raised cap" is very common.

Command \init{letter}[some text] does this:

  • A blank line is inserted, so that the following "raised cap" does not bump into anything above it.
  • The letter is scaled to 1.5x its normal size. If the letter has a descender (JQÇ¡¿) then Novelette automatically adjusts the descender, so that it does not bump into the following line of text.
  • The optional some text will be printed in small caps (only lowercase). Do not attempt to include the entire first line of text. Note that some text does not duplicate the initial letter:
    \init{D}[uring the day], they slept.
  • If main text begins with punctuation (quotes, apostrophe, guillemets, Spanish inverted marks), then put the punctuation alongside letter. It will automatically be adjusted to a compatible style, depending on the punctuation. Note: Although there are many possible styles, Novelette uses the style that appears to be most common. Example:
    \init{``¿Q}[ue pasa?''] dit Juan.

    • In some cases, you may wish to adjust how letter relates to some text. Use \kernvalueem where value is a number (usually small negative):
    \init{T\kern-.04em}[o be], or not to be.

  • The \init does not need to be the first item on its line. Some books use multiple indents, so that the start of text begins mid-line. If you wish to do that, use several \forceindent before \init.

\scene{text}

If a chapter has scenes, then you must separate them, using some kind of visual cue.

You may use one or two \null to insert blank lines. Then the next scene must either begin with \noindent or \init.

Another method is the \scene{text} command. This has more flexibility:

  • If text is empty, then two lines are skipped. Positioned midway between the scenes, three widely-spaced asterisks are centered.
  • If text is an asterisk, the above applies.
  • If text
    • is a hyphen, then instead of asterisks, a short horizontal line is centered.
  • If text is anything else, then an additional line skip is inserted. Total three line skip. Slightly below the middle line, the text is printed, no indent, at slightly larger size.
  • If the scene break happens to occur at a page break, then one of the blank lines will be omitted (because it is not needed there). However, you should check this yourself.
  • In all cases, the start of the following scene is not indented.
  • You may begin the following scene with \init, but this may be visually cluttered unless the line gap is empty.
