File Organization

Novelette has built-in support for sub-documents. This allows you to create a hierarchy of folders and files, which are components of the main document. Then, you may compile the entire main document with all of its components, or compile an individual component. This is very useful for writing and editing.

Read about sub documents on another page. It will also direct you to the included demonstration files.

Note that Novelette does not use the \include or \import commands for this purpose.

No Packages!

Novelette has all support (including fonts) built-in. You may not use \usepackage or \RequirePackage to add some other capabilities.

Why not? The precision layout is very likely to be wrecked, if any other packages are used. Also, other packages amy cause PDF/X conformance to fail, yet Novelette could not detect the failure.

Preamble Settings

Novelette does not use class options to set page size, margins, fonts, or any other dimensions. Instead, each available setting is in Preamble.
