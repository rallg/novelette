Novelette - layout styles

Novelette offers seven page payout styles (plus style=0 for developer use). These are chosen in Preamble using \layout{style=N} where N is the style number.

  • style=0 provides no header or footer. For developer use.
  • style=1 provides only footer, with centered page number.
  • style=2 provides header and footer, with centered text and page number. This is the default style.
  • style=3 provides header and footer, with text and page number aligned to the outside margin. This style is unusual.
  • style=4 provides only header, with centered text, and page number aligned to the outside margin.
  • style=5 is like 4, with added vertical bar at fixed distance from the page number. This style is frequently used.
  • style=6 is like 4, but with added slash next to the page number.
  • style=7 provides only header, with text and page number both centered, separated by a slash. The number is toward the outside.
