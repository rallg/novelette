Novelette - layout schemes

Novelette offers seven page layout schemes (plus scheme=0 for developer use). These are chosen in Preamble using \layout{scheme=S} where S is the scheme number.

  • scheme=0 provides no header or footer. For developer use.
  • scheme=1 provides only footer, with centered page number.
  • scheme=2 provides header and footer, with centered text and page number. This is the default style.
  • scheme=3 provides header and footer, with text and page number aligned to the outside margin. This style is unusual.
  • scheme=4 provides only header, with centered text, and page number aligned to the outside margin.
  • scheme=5 is like 4, with added vertical bar at fixed distance from the page number. This style is frequently used.
  • scheme=6 is like 4, but with added slash next to the page number.
  • scheme=7 provides only header, with text and page number both centered, separated by a slash. The number is toward the outside.
