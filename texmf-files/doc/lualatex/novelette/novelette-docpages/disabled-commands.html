Disabled Commands

Novelette disables the use of many LaTeX commands or environments. This page lists a number of them. There may be others, but LaTeX has numerous commands, many of them specific to certain packages, so it is not possible to list them all.

Math Mode - The dollar symbol ($) prints as dollar symbol. It does not invoke math mode. Other methods for entering math mode are also disabled.

Environments: list, equation, table, XXXXX are all disabled.

Text effects: bold, underline, strikeout, color, are disabled.

Size commands such as \small and \large are all disabled. You may use \smalltext in fullpage, legal, and upperpage environments. You may scale text (relative to normal size) in \name, \subname, and \formatX commands.

Graphics: \includegraphics is disabled. Use \image or \icon instead. You cannot use \caption. You cannot apply image transforms such as reflection or scale. You cannot use \graphicspath, because the path to any image or icon is always relative to the main document (even if requested by subdoc),

Environments: Novelette only recognizes the environments described here.

Among the standard LaTeX environments not recognized: abstract, array, description, displaymath, enumerate, eqnarray, equation, figure, itemize, labeling, list, math, minipage, picture, quotation, quote, tabbing, table, tabular, theorem, titlepage, verbatim, verse.

Commands: Novelette does not recognize many standard LaTeX commands. Among them: \part, \chapter, \section, \subsection, \subsubsection, \paragraph, \subparagraph, \appendix, \makeindex, \maketitle, \theindex, \tableofcontents, \listoftables, \listoffigures, \caption, \index.

Commands involving bold, underline, strikethrough are disabled.
