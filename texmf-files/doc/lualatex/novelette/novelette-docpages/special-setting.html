In General

This page lists a few Preamble settings that are used less often. Only use these settings if you need them, and understand what they do.

top

\headers{verso header text}{recto header text}

Default header text is the author's name (Italics) on verso pages, the book title (all small caps) on recto pages. You may change this using \headers.

The header texts may be styled, using \textit{text} for Italics, \smcp{text>} for lowercase to small caps, and \allsmcp{text} for both uppercase and lowercase to small caps. No other styles are honored.

If you choose font=light or font=dark in \layout, then requests for Italics or small caps will be ignored, because those fonts do not have them.

Unless you have good reason, it is best to use the default headers. The reason is that print-on-demand discourages (sometimes rejects) the use of anything but author and title in page header text. This is because the manufacturer is unfamiliar with your book.

You cannot change the header text mid-book. It is a global setting.

What if your book is "two books in one"? Do not change the headers. Give your book a title that does not commit to one part-book or the other.

\enable{setting}

There are a few things that Novelette can do, but by default does not do them. This is because they use extra compile time, or require files that many users might not have. Why waste time, or throw errors, trying to do something that you do not need?

The only user-level setting is \enable{alt}. This loads the Libertinus Serif fonts, and allows you to use the \alt{char} command in your document.

Other settings (not documented) are for developer use.

top

\myparindent{length with units em}

The default, global paragraph indent is 1em. If you need somthing else for the global indent, this sets it. Be sure to specify the size in units of em, rather than points. Minimum 0.8em.

Local changes to paragraph indent are automatically handled by macros. If they are incorrect in a particular situation, it is better to use \forceindent or \backindent for one-time fix.

\sheetfeed{letter or A4}

Do not use the \sheetfeed setting, unless you are certain that you need it. When you do not set it, Novelette will automatically create PDF page size equal to the book trim size. This is how most manufacturers want it.

If you are printing copies on your home printer, perhaps because you like to edit with pen and ink: The printer may not know what to do, because its paper is not the same as the requested print size (trimsize). In this case, set sheetfeed to match the paper, either letter or A4. That will center the print area in the paper.

Be sure that you do not use \sheetfeed for the final file sent to the manufacturer, unless requested.

In final mode, using PDF/X (any \intent other than none), the PDF will have an invisible TrimBox (part of PDF/X standards). This tells software that the PDF paper size (sheetfeed) is not the same as the finished size (trimsize).
