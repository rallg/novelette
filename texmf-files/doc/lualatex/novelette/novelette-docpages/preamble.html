Preamble (Book Interior)

This page lists all Preamble settings for the book interior pages. For information regarding cover artwork, go here.

Minimal Preamble

Almost everything in Novelette has a pre-defined default setting. You may be able to write your book without learning most commands.

This Preamble is enough to create a minimal working example. The PDF will be in draft mode, which has correct typesetting but is internally marked as draft.

\documentclass{novelette} % Use lualatex only.
\trimsize{w=width,h=height} % Required. Try {w=5.5in,h=8.5in}
\begin{document}
Write text here.
\end{document}

Complete Preamble for Book Interior

Novelette defines a number of Preamble settings. Most of these settings have reasonable defaults, so you do not need to set all of them.

You must set \trimsize. This is the "trim size" of the book interior pages. There is no default.

Note that you cannot use \usepackage or \RequirePackage to add more packages. Novelette is very restrictive, because added packages are likely to break other settings. The packages you need are already included. If you wish to add something else, then either do without it, or use a different document class.

Here is a list of all available Preamble settings, for book interior. Click a setting, and you will navigate to its HTML documentation:

% !TeX TS-program = lualatex
% !TeX encoding = UTF-8
\documentclass{novelette}
\title{Title of Book}
\subtitle{Subitle of Book} % Optional.
\mode{setting}
\intent{setting}
\application{setting} % Optional.
\producer{setting} % Optional.
\language{language}[option]
\trimsize{setting} % Required.
\sheetfeed{setting} % Rarely used.
\margins{setting}
\typesize{setting}
\lines{setting}
\layout{setting}
\guides{setting}
\fallback{enabled|disabled}
\titlefont{setting}[options]
\titlefont{setting}[options]
\decofont{setting}[options]
\namestyle{setting}
\subnamestyle{setting}
\microtype{setting}
\myparindent{setting}
\warnalert{t|f}
\begin{document} % End of Preamble.
Your writing is here.
\end{document}

Magic Comments

Although they are not required, it is helpful to write these two lines at the very top of your document (including the % symbol):

% !TeX TS-program = lualatex
% !TeX encoding = UTF-8

Those are "magic comments" that are understood by some GUI programs, such as TeXworks. They tell the program how to compile your document.

If you are compiling from command line, the magic comments are ignored.

Document Class

With the exception of cover artwork, there are no document class options. All settings are performed in Preamble using the commands described here. This behavior is very different from standard LaTeX document classes (such as article or memoir).

\warnalert{t|f}

Novelette detects a few situations where you did something unusual, but not bad enough to deserve a Warning or Error. Them, it writes an ALERT message in the log file. If there was at least one alert, there will also be a message in the log summary.

Unless there was another problem, a request for final mode is honored, and the PDF will print. Depending on what you are trying to do, some alerts may be acceptable, or not. This is for you to decide.

Some GUI programs (such as TeXworks) can be configured to display a list of Errors, Warnings, and bad boxes. But they do not detect an alert. If you write \warnalert{t} (true) in Preamble, then Novelette will issue a Warning when there is at least one alert. That will be detected by the GUI.

Default is \warnalert{f} (false).
