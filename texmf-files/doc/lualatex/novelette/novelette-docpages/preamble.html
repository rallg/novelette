Interior Preamble

Terminology

The main document or top-level document is your book interior. Everything from the top of its file, up to and including \begin{document}, is the Preamble. The Preamble contains various settings and other descriptive information.

Following \begin{document}, up to and including \end{document} is the document body, containing the printable content of your book.

The document body may be (and should be) divided into subfiles, which allow you to work on portions of the book.

Complete Preamble

Unlike other TeX document classes, Novelette does not allow you to add other code (packages). It is self-contained. As a result, there are only a limited number of possible settings in Preamble.

The only required setting is \trimsize, even in draft mode. For final mode, you will also need title and author. Every other setting has default values, chosen for the purpose of popular fiction.

Each setting is described in detail, on various documentation pages. Click a setting to learn more.

% !TeX TS-program = lualatex
% !TeX encoding = UTF-8
\documentclass{novelette} % No options, except for cover.
\title{Title of Book}
\subtitle{Optional Subtitle}
\author{Author of Book} % May be pseudonym.
\mode{setting}
\intent{setting}
\trimsize{setting} % Always required.
\margins{setting}
\layout{setting}
\namestyle{setting}
\subnamestyle{setting}
\formatastyle{setting}
\formatbstyle{setting}
\formatcstyle{setting}
\formatdstyle{setting}
\formatestyle{setting}
\formatfstyle{setting}
\guides{setting}
\typoone{setting}
\typotwo{setting}
\typotwo*{setting}
% Rarely-used settings:
\sheetfeed{setting}
\myparindent{setting}
\myroman{setting}
\begin{document}
