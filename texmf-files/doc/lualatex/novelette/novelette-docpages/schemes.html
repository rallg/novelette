In General

These illustrate a page spread: verso at left, recto at right.

Schemes 2 and 3 are among the most widely-used in popular fiction. Since scheme 2 uses more vertical space, it is less frequent in smaller books.

If your layout uses a header, its text must be brief enough to fit within the available horizontal space. It must be only one line.

scheme 3
Header text centered.
Page number at outside margin.
Page icon at fixed distance from outside margin.
Default page icon is vertical bar.
No footer.

scheme 4
Centered header.
Verso: page number, page icon, text.
Recto: text, page icon, page number.
Default page icon is slash.
No footer.
