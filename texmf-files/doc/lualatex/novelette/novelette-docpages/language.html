Language Settings

Novelette allows only one language choice. You may write the occasional word and phrase in any language, but if it needs hyphenation, it will be hyphenated according to the rules of your single language.

During the Alpha stage of Novelette, language selection is subject to change. In particular, the correct use of French punctuation spacing, and language-specific quotation marks, is under development.

Novelette does not load either babel or polyglossia packages, although it does use some code from each of them.

Your language must be Western European, although a few languages that use the same character set (such as Finnish) are supported.

Command: \setlang{language}[option]

The language is lowercase, and may be any of english, french, german, and many others. Most languages do not have an option. In the case of english, use option [us] or [uk] for best results.

Note that the command is \setlang. If you attempt to use something else, such as \lang or \language or \setlanguage, then an error results.

Novelette will use hyphenation, if the support package for your language is installed. It will NOT format dates and times, or make other style changes.
