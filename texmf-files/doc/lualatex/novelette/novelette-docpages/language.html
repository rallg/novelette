Language and Smart Quotes

In order to do one thing well, instead of many things, Novelette is limited to several major Western European languages. If your language is not one of these, then Novelette does not support it: English French German Italian Spanish. Some have options.

You may write some words or phrases in another language. However, they will be processed according to the rules for the single, main language.

\setlang{language}[option]

This command is used in Preamble. Once a language is set, it cannot be changed. If you do not set a language, then Novelette will use a default, which is suitable only for testing and examples.

Languages and options are written as lowercase, using the English spelling (french, not French or francais or français).

  • \setlang{italian} has no options
  • \setlang{spanish} has no options.
  • \setlang{english}[option] has two options:
    [us] for American spelling (red is a color).
    [uk] for British spelling (red is a colour).
    If you do not set either option, then an older, generic American is used. But the generic is out-of-date. Be sure to use an option.
  • \setlang{french}[option] has one option:
    [swiss] writes less space before the colon (:).
    With or without option, French punctuation spacing is used.
  • \setlang{german}[option] has five options:
    [1901] traditional German (with eszett).
    [1996] neue German (with ss).
    [1901x] like [1901], improved.
    [1996x] like [1996], improved.
    [swiss] Swiss spelling.
    Prefer [1901x] to [1901], and prefer [1996x] to [1996]. The preferred options require package dehyph-exptl installed.
  • There is no need for other regional variants. For example, the rules for Mexican Spanish, are the same as for Spain. This is because Novelette only processes hyphenation, smart quotes, and (in the case of french) punctuation spacing. Regional variants, such as date/time format, or money, are not automated.
  • You may request \setlang{none} for no language support.

Smart Quotes

Novelette uses the built-in luaquotes package, for smart quotes. When you type ordinary single (') and double (") quotes, they will be replaced by matching left and right quotation marks, appropriate for your language.

For example, suppose you type 'this' and "that". They will be converted:

  • For english, italian, spanish: ‘this’ and “that”.
  • For french (including swiss): ‹ this › and « that » with extra spacing.
  • For german (not swiss): ‚this‛ and „that‟.
  • For german swiss: ‹this› and «that».
  • You may globally over-ride the above defaults. See next section.
  • You may also paste characters from a character map, or from a keyboard that has language-specific quotation marks.
  • Novelette also allows "TeX ligature" quotation marks. If you do not know what that means, ignore it.
  • If a word begins with apostrophe (omitted letter), then it will be mis-interpreted as left quote. Examples in english: 'bout (about), 'til (until). Use \apost there: \apost bout and \apost til.

\luaquotestyle{choice}

This Preamble setting is optional. Only use it when you wish to over-ride the default smart quotes. Choices:

  • off disables smart quotes.
  • en uses quote style of english, spanish, italian.
  • fr uses quote style of french (with spacing).
  • de uses quote style of german (not swiss).
  • dech uses quote style of german swiss. Sometimes used in other languages.
  • degm uses an alternate german quote style: ›this‹ and »that«.
  • You may always place any style of quotes, by writing the exact characters.
