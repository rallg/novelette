Terminology

Novelette uses layout to mean page design: Is there a running header? A running footer? How much vertical space between main text area and the header/footer? Does the text in header/footer look similar to main text, or does it have a very different style?

When layout includes header/footer, that space is reserved on every page, even when a particular page has empty header or footer.

You do not have full control over layout. Instead, Novelette forces you to choose from a limited selection. Why? There is less for you to think about (so you have more time to write your story). Good news: Just about every book, in the popular fiction shelves of public libraries, uses these designs. Novelette knows what to do!

\layout{scheme=S,gap=G,lines=L,em=M,font=F}

You may set all, some, or none of \layout. Novelette provides a default value for anything you do not set.

  • scheme=S : S is an integer 1-8. It chooses whether layout has header and/or footer, and alignment of header content. The choices are illustrated in the gallery of layout schemes. Default is scheme=2. Another popular choice is 5.
    • scheme=1 : Centered footer, no header.
    • scheme=2 : Centered header and footer.
    • scheme=3 : Header and footer, aligned outside.
    • scheme=4 : No footer. Centered header text. Number outside.
    • scheme=5 : Like 4, with vertical bar near page number.
    • scheme=6 : Like 4, with slash at page number.
    • scheme=7 : No footer. Centered header text/number.
    • scheme=8 : Like 4, with icon instead of bar.
  • gap=G : G is a number from .5 to 1. May be any intermediate value. This sets extra vertical space between header (if used) and main text, and between main text and footer(if used). The space is measured in lines. Value .5 inserts a half-high empty line. Value 1 inserts a full empty line. Default gap=1 is good, but you may prefer a smaller value for a small trimsize.
  • lines=L is integer lines per page, main text block (not counting header/footer). Typical value is in range 27-35.
  • em=M is the em size of main text, with units pt or bp. May be decimal (and usually is). Typical value is in the range 11.4-11.7pt.
  • font=F : F is one of main,light,dark. Default main. This sets whether a stylish font is used for page headers. It does not affect main text.
    • font=main (default) uses Swainson, same as main text, but slightly tweaked for used in page headers. This is the only choice that provides both Regular and Italic, also with small caps.
    • font=light uses Towhee-Light. Only Regular, no small caps. However, lowercase letters appear as reduced-size uppercase.
    • font=dark uses Towhee-Dark. All letters appear uppercase, with distinctive appearance.

\guides{setting}

You may request up to four pairs of guides. These are marks that appear in the side margins. They are only visible with \mode{shade}

The setting is four or less comma-separated integers. Each integer is a line number where the guide pair will appear. Line 1 is the top line of main text (not including header). For example, \guides{5,14} places guides at lines 5 and 14.

Each guide is a small rectangle. The bottom of the rectangle is at a main text baseline. The height of the rectangle is the x-height of main text.

When you start chapters with the upperpage environment, it has no particular height. Use guides to position things so that each chapter presents a uniform appearance.

Guides are global. They are not enforced. That is, nothing "snaps to" a guide. Whether or not to obey them, is for you to decide.
