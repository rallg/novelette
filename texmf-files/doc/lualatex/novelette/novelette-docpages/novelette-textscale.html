Text Scale

This page does not discuss cover artwork.

Command: \textscale[position]{scale}{text}

Novelette does not allow use of standard LaTeX size commands such as \small and \Large. If you use any of those commands, they will simply be ignored. This is necessary to preserve the line grid.

On a display page, and in the opening environment, you may use the \textscale command. It applies to a single line of text, without wrapping. If you need multiple lines of scaled text, then you must use multiple commands, one for each line.

Within \textscale you may change the font, use Italics, and select OpenType features. But you may not nest another textscale, and you may not place an image.

This command is used for creating prominent text, such as on the book title page, and in chapter headings.

  • [position] is one of l, c, r, o, i. This positions the text left, center, right, outside, inside. Default c.
  • {scale} is a number from 0.8 to 6. May be decimal. It is the scale of the text, relative to the size of main text.
  • {text} is the single line of text, to be scaled. Do not end it with \par, because it is already included.

The scaled text will probably not have its baseline on the line grid. However, the baseline is restored after scale, so any following main text will be on line grid. This is especially important when \textscale is used in chapter openings.
