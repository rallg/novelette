Various Text Commands

This page is under construction. It will describe various commands that are useful within the main text body.

\ital{Text in Italics font.}

This is an alias for the LaTeX \textit command. If used within a paragraph, there will be a small amount of space added after the final character, to allow for slant. You may enclose more than one full or partial paragraph. Also see the \emph command.

\emph{Emphasized text, Italics or not.}

This is like \ital, but: If the surrounding text is already in Italics, then \emph removes the Italics from its text.

\bold{Text in bold font, maybe.}

Novelette will ignore this command, and print the unmodified text, if you request \bold in the main body or header/footer. This is because popular fiction would never use bold there.

If the command is written within an opening environment, or on a display page, then the text will be bold.

\small, \large, and similar commands do not work

Surprise! LaTeX size commands do not work in Novelette. This is intentional. Popular fiction does not change its text size, except in special locations. If you use any of these commands, the text will be printed without size change.

Using different commands, size changes are allowed in the opening environment and on display pages.

\feat{code}{text}

This command adds and/or removes OpenType features from text, where code is one or more comma-separated, 4-character OpenType features. If a feature is negated, it is removed. Otherwise it is added. Examples:

  • \feat{-onum,cv14}{text} removes onum and adds cv14 to the text.
  • \feat{ss02,cv24} adds both ss02 and cv24 to the text.
  • \feat{}{text} retains existing text features. This is useful as a placeholder, when you are not sure which features to use, because they vary from font to font.
  • Some features (such as onum and lnum} conflict. To ensure consistent results, remove one feature, before adding the other.
  • If you attempt to add a feature that is not in the font, or remove a feature that was not set, Novelette will not complain. However, the full log file will have messages from the fontspec package.

Short-Form Feature Commands

Some OpenType features are often added or removed. For your convenience, here are short forms. None of these affect features such as style sets ssNN or character variants cvNN.

  • \onum{text} removes any other number style, then adds onum. It does not affect letters.
  • \lnum{text} removes any other number style, then adds lnum. It does not affect letters.
  • \smcp{text} removes any other letter style, then adds smcp. It does not affect numbers or uppercase letters.
  • \allsmall{text} combines smcp and c2sc and onum.
