In General

This page lists a few Preamble settings that are almost never used. That is, Novelette has default values, which are almost always best. Only use these settings if you need them, and understand what they do.

top

\myparindent{length}

The default, global paragraph indent is 1em. If you need somthing else for the global indent, this sets it. The length may be measured in pt or other units, but it is best to specify with em. The value must be at least 0.8em. If you provide a setting that does not make sense, or is too small, Novelette will ignore it, issue an alert, and use 1em.

Local changes to paragraph indent are automatically handled by macros. If they are incorrect in a particular situation, it is better to use \forceindent or \backindent for one-time fix.

top

\myroman{i or I}

The default setting is i. If any page numbers print in frontmatter, they will be lowercase roman numerals. Setting I is for purists who know that the Romans did not have lowercase, so this uses uppercase instead.

However, your book probably has no printed page numbers in frontmatter, so \myroman is usually unimportant.

This setting does not affect arabic page numbers in mainmatter.

\sheetfeed{letter or A4}

Do not use the \sheetfeed setting, unless you are certain that you need it. When you do not set it, Novelette will automatically create PDF page size equal to the book trim size. This is how most manufacturers want it.

If you are printing copies on your home printer, perhaps because you like to edit with pen and ink: The printer may not know what to do, because its paper is not the same as the requested print size (trimsize). In this case, set sheetfeed to match the paper, either letter or A4. That will center the print area in the paper.

Be sure that you do not use \sheetfeed for the final file sent to the manufacturer, unless requested.

In final mode, using PDF/X (any \intent other than none), the PDF will have an invisible TrimBox (part of PDF/X standards). This tells software that the PDF size (sheetfeed) is not the same as the finished size (trimsize).
