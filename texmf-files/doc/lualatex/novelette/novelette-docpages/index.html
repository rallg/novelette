Minimal Preamble

Almost everything in Novelette has a pre-defined default setting. You may be able to write your book without learning most commands.

This is the minimal document Preamble, enough to create a minimal working example. The PDF will be in draft mode, which has correct typesetting but is internally marked as draft:

\documentclass{novelette}
\booksize{w=width,h=height} % Required.
\begin{document}
Contents of your book go here.
\end{document}

Complete Preamble

Novelette defines a number of Preamble settings. Most of these settings have reasonable defaults, so you do not need to set all of them.

You must set \booksize. This is the "trim size" of the book interior pages. There is no default.

Note that you cannot use \usepackage or \RequirePackage to add more packages. Novelette is very restrictive, because added packages are likely to break other settings. The packages you need are already included. If you wish to add something else, then either do without it, or use a different document class.

Here is a list of all available Preamble settings, for book interior. Click a setting, and you will navigate to its HTML documentation:

% !TeX TS-program = lualatex
% !TeX encoding = UTF-8
\documentclass{novelette}
% Metadata:
\title{Title of Book
\author{Author of Book}
\subtitle{Subtitle of Book}
\mode{setting}
\intent{setting}
\application{setting}
\producer{setting}
% Book and page layout:
\booksize{setting} % Required.
\sheetfeed{setting} % Rarely used.
\margins{setting}
\typesize{setting}
\lines{setting}
\layout{setting}
\guides{setting}
% Choice of Fonts: \mainfont{setting}[options]
\sansfont{setting}[options]
\monofont{setting}[options]
\altfont{setting}[options]
\dramafont{setting}[options]
\decofont{setting}[options]
% Text Adjustments: \namestyle{setting}
\subnamestyle{setting}
\setmicrotype{setting}
\myparindent{setting}
% Special Purpose: \warnpercent{setting}
\warnquotes{setting}
\warnalert{setting}
\begin{document} % End of Preamble.
Commands that may be used in the document body are on another page.
\end{document}

Description of Preamble Settings

Magic Comments

Although they are not required, it is helpful to write these two lines at the very top of your document (including the % symbol):

% !TeX TS-program = lualatex
% !TeX encoding = UTF-8

Those are "magic comments" that are understood by some GUI programs, such as TeXworks. They tell the program how to compile your document.

If you are compiling from command line, the magic comments are ignored.

^return

Document Class

With the exception of cover artwork, there are no document class options. All settings are performed in Preamble using the commands described here. This behavior is very different from standard LaTeX document classes (such as article or memoir).

^return

Metadata

Metadata includes title, author, mode, and intent. Rarely, you may set application and/or producer.

Metadata is read by PDF software, and the printer. But it does not print. In particular, Novelette does not automatically create a title page from your title and author settings. You must do it yourself.

The settings are described here.

^return

Book Size and Page Layout

The "book size" is Novelette's term for "trim size", which is the interior page width and height. You must set it. There is no default.

How to set book size, margins, type size, and other aspects of interior page layout, are explained here.

Font Selection

Novelette does not load fonts by \usepackage or \RequirePackage commands. It relies on the built-in fontspec technology.

However, fontspec does not load until after Preamble. Novelette has several font settings that temporarily store the information, then pass it to fontspec later.

Only a small number of font settings are accepted. They are described in detail on another page.

^return

\namestyle{align=A,font=F,scale=S}

The \namestyle setting provides a uniform appearance for text that uses the \name command. Example here.

  • align=code determines the horizontal alignment. The code is one of l c r o i (left, center, right, outside inside). Default align=c.
  • font=code determines which font will be used for the name text. The code is one of main sans mono alt drama deco. Default font=main.
  • scale=number sets the scale of name text, relative to main font size. Default scale=1.6, which looks nice in chapter names.

^return

\subnamestyle{align=A,font=F,scale=S}

The \subnamestyle setting provides a uniform appearance for text that uses the \subname command. Example here.

The use of \subnamestyle is like that of \namestyle, except that this default scale=1.25.

^return

\setmicrotype{settings}

Novelette uses the microtype package for improved appearance of body text. Since the package does not load until after Preamble, this command provides a way to "PassOptionsToPackage" when microtype loads.

If you do not know what that means, simply ignore it. Novelette is pre-set to values that are good for its purpose. You do not need this command.

If you do wish to play with this command, Novelette will not let you use the microtype draft mode. Microtype will always be in its own final mode, no matter what you choose via \mode in Novelette.

^return

\myparindent{size}

Novelette pre-sets the default paragraph indent at 1.5em. That looks good for the purpose intended. But if you need a different default value, use \myparindent.

  • Example: \myparindent{10pt} for a small indent.
  • This is a global setting. If you need to locally change the indent, that is performed using other commands in body text.

^return

\warnpercent{on or off}

TeX uses the percent character % for comments. Anything following %, on the same line of raw text (prior to typesetting) will be ignored.

If you need to print the percent character, you may use a backslash \% there.

Default \warnpercent{off} does nothing. But \warnpercent{on} issues a Warning if you use the un-escaped % anywhere in body text. It does not warn for Preamble. This is a way to detect accidental use of % (which means comment) when you really meant to print %.

In body text, you may use \memo{This does not print.} where you need it. The difference between memo and %, is that after the memo completes, anything following it will print.

If you use this to issue a Warning, then the PDF will not be in final mode.

The novelette-docpages directory has a file named "warnmebro.tex". You may edit it with warnpercent on or off, for comparison.

^return

\warnquotes{on or off}

TeX normally converts straight double quotes " to right curly quotes ‚Äù. Alas, some writers (moi) are in the habit of typing " with the expectation that a smart-quotes feature will alternately convert them to left and right curly quotes, as needed. That is not what TeX does, so the print will be wrong. This is often hard to notice.

Default is \warnquotes{off}. If you use \warnquotes{on} then Novelette will issue a Warning if you wrote " anywhere in the document body. It does not warn for Preamble.

Since Novelette uses UTF-8 text encoding, you may directly paste left and right curly quotes from a character map. Or, you may use `` (two acute characters, also known as backticks) for left curly quotes, and two '' (straight single quotes, also known as straight apostrophe) for right curly quotes.

If you use this to issue a Warning, then the PDF will not be in final mode.

This documentation directory has a file named "warnmebro.tex". You may edit it with warnquotes on or off, for comparison.

^return

\warnalert{on or off}

Novelette sometimes writes ALERT messages to the log file. These are not Errors, or Warnings. Usually, an ALERT means that you set something to an unusual value. Did you really mean that, or not?

Default \warnalert{off}. If you use \warnalert{on} then a Warning will be issued if there were any Novelette-caused ALERTs.

It may be that the ALERT was thrown by a situation that you should fix. Or maybe it is what you intended, so no need to fix it. When you are satisfied that nothing needs to be fixed, turn \warnalert{off}.

If you use this to issue a Warning, then the PDF will not be in final mode.

This documentation directory has a file named "warnmebro.tex". You may edit it with warnalert on or off, for comparison.

^return
