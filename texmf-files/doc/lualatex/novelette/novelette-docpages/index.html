\\ (double backslash) causes a line break. The line is not justified. Might be useful with \style, or with poetry, but generally not used in main text. See also \par and \linebreak.

\allsmcp{text} styles both uppercase and lowercase letters as small caps, and numerals as oldstyle. These must be in the font; they are not faked.

\alt{text} uses \altfont for the text.

\altfont{Font Name}{features} specifies a font intended as a "fallback", to provide characters not in the main font. The substitution is not automatic, but must be manually coded using \alt{text}.

\application{setting} rarely-used Preamble setting. Changes the PDF metadata Application from Novelette to whatever you set.

\author{Author Name} required for final mode.

\backindent if a paragraph indent appears where you do not want it, this command removes it.

Block Indent (topic).

\bold{text} is only effective on display pages, and in chapter openings. Otherwise, it will be ignored.

\booksize{width,height} is the "trim size" of your book interior pages. This setting is required. There is no default.

Chapters and Scenes (topic)

\clearpage finishes the current page, so that following content begins on a new page. Consecutive \clearpage does not skip a page. You may use \blankpage for that.

\cleartorecto finishes the current page. Following content begins on a new page. If necessary, a blank page is inserted, so that following content is recto.

\cleartoverso finishes the current page. Following content begins on a new page. If necessary, a blank page is inserted, so that following content is verso.

\coverimage{filename.jpg} specified the cover image, if you use Novelette to convert it to PDF/X. The image must already be CMYK, pre-processed using script cmyk4nvt.

\covertrimsize{width,height} is the finished cover size (rear, spine, front). It is smaller than the coverimage size, by the bleed width on all sides.

\lines{integer} is a Preamble setting. Lines per page in main text, not including header/footer. If omitted, Novelette will calculate a nice value.

\lnum{text} writes included numbers with lining numerals, if they are in the font.

\noindent requests no paragraph indent. Also see \backindent.

\null inserts a blank line. This command may only be used between complete paragraphs or images.

\onum{text} writes included numbers using oldstyle numerals, if they are in the font.

\typesize{size} Preamble setting, for the size of normal main text, using units of TeX points (pt) or PostScipt points (bp). May be decimal. Typical size is about 11.7pt. If unset, Novelette will guess a likely size.

\uni{HHHH} where HHHH is four hex digits 0-F. Writes the Unicode character at HHHH in the basic multilingual plane.

\versohead{text}

\vfil and \vfill

\warnalert{on|off} Preamble setting. Default off. If on, then Novelette alerts will be converted to warnings, which are detected by some GUI programs. Also, the warnings will disallow final mode. Note that many alerts may be safely disregarded.
