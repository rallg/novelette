Basic Document Structure

The top three lines of your document look like this:

  • % !TeX TS-program = lualatex
    % !TeX encoding = UTF-8
    \documentclass{novelette}

The first two lines are "magic comments" that tell some GUI programs (such as TeXworks) how to process the document. They are ignored when you process directly by command line.

Except when processing cover artwork, Novelette has no document class options. Sizes and layout are selected by settings in Preamble. These are listed below.

Settings in Preamble

Every document, even tests, must set the book size. There is no default value.

Every book requires title and author. If you do not provide both of these, your document will be interpreted as a test file.

There are several processing modes. Default mode is draft.

If you need PDF/X conformance, then choose an Output Intent. Default is none, which does not conform to PDF/X.

Every printer requires margins set to at least some minimum width. If you do not set margins, Novelette will provide commonly-used values as defaults.

Choose a page layout to your taste. This determines whether or not there will be headers and/or footers, and how they will appear. If you do not set layout, Novelette provides a default.

Novelette comes with its own Swainson and Towhee OpenType fonts, which are specifically designed to look good in popular fiction. They are used everywhere, by default. But if you wish to change the fonts, or add more fonts for special purposes, here is how.

After it reviews your layout, Novelette will automatically choose suitable values for main font size, and lines per page. If you want to use something else, you may manually set the font size and lines.

Each chapter (and related content, such as Prologue) should start with a uniform appearance, with chapter name and (possibly) subname, above the start of main text. If you do not set this, Novelette will use defaults.

When you are using shade mode, Novelette can display guides in the side margins.

Novelette uses the microtype package for enhanced typesetting. You may change its configuration, but do not change it unless you have expertise.
