Interior Page Fonts

Fonts: No Choice!

Unlike most other LaTeX document classes, Novelette does not choose fonts via \usepackage{Font Name}. It does not set fonts by \setmainfont or \rmfamily or any other command.

Main Font : Swainson

Whether you like it or not, the main font is always Swainson, which comes with Novelette. This font has Regular and Italic. Its character set includes Latin-1, typographic support, small caps, lining/oldstyle numerals, and a few other characters beyond Latin-1. In other words, it is well-suited to the limited number of languages that Novelette supports. By using a specific font with known properties, Novelette can do some things more accurately than would otherwise be possible with arbitrary fonts.

Popular fiction has no need of bold fonts. Swainson "Bold" is actually a copy of Regular, and "BoldItalic" is actually a copy of Italic. This trick is for the purpose of keeping other software satisfied, in the event that you work with an Editor who is not using TeX.

Swainson is vaguely in the category of Bembo/Caslon/Garamond fonts, but is not a derivative or reproduction of any of them. It as designed using the LaTeX Cochineal/Crimson fonts as reference models, but the glyph shapes are noticeably different.

Swainson is released under the terms of the SIL Open Font License, v1.1, and thus is suitable for commercial usage. The font is named after Swainson's Thrush, a seasonal songbird.

Stylistic Font : Towhee

Swainson is the only font for main text (and footnotes), and is the default font everywhere. In special places, such as page headers, chapter openings, and title page, you may choose a stylistic font.

Novelette provides several Towhee fonts. These are not related, other than by name. They are selected by choosing "font" as option to one or more of these commands: \name, \subname, \formatX (X = a, b, c, d, e, f) and \layout.

Towhee fonts are released under the terms of the SIL Open Font License, v1.1, and thus are suitable for commercial usage. The fonts are named after the Spotted Towhee, a seasonal songbird.

  • Towhee-Regular is invoked by font=sans. It is similar to Swainson-Regular, without serifs. Includes small caps, but not Italic.
  • Towhee-Light is invoked by font=light. It does not have small caps or Italic. Its lowercase letters are reduced-height uppercase. This shape is particularly suitable for larger sizes, such as title page.
  • Towhee-Medium is invoked by font=dark. It has only uppercase letters (typing lowercase will print uppercase). Its style is very different from Swainson. It is particularly suitable for dramatic page headers.

Fallback Font: Libertinus Serif

The Swainson character set is intentionally limited to Latin-1, plus typographic support, plus a few other characters outside Latin-1. This suffices for the intended languages of English, French, Catalan, Spanish, Italian, and German.

Eastern European, Greek, Cyrillic, and other world languages are not supported. However, if you need the occasional character for a word or phrase, you may use a fallback font. This is not automatic. You must do several things to make it work:

  • Use the TeX package manager, to install libertinus-fonts. This enables Libertinus Serif as fallback font, Regular and Italic. It has a very large character set, and is automatically tweaked so that its characters blend well with Swainson. Its license is SIL OFL v1.1.
  • In your Novelette document, command \alt{char} to print char in the Libertinus Serif font. The char may be one or more utf-8 text characters, Regular or Italic.
  • If you use \alt but did not install libertinus-fonts, an error results.
  • You cannot use \alt within \title, or \author, or within page headers/footers. These are places where you should not be using fallback characters, so it is not a real limitation.
  • If the fallback characters are modified (Italics, small caps, or other features), write the modification inside the \alt command. Wrong: \ital{\alt{char}}. Correct: \alt{\ital{char}}.
  • Do not write \alt within stylistic text (Towhee). It will look wrong.

Need Something Dramatic?

The book title page (and perhaps half-title) may benefit from using an image, instead of text. This can create powerful visual effects that connot be easily achieved by fonts, even if Novelette allowed more fonts.

If you use images, be sure that the decorative title will read the same as text. That is, do not create an image so fancy that its letters are no longer obvious.

In all cases, the copyight page must have the book title in ordinary text.
