Fonts

Unlike most other LaTeX document classes, Novelette does not choose fonts via \usepackage{Font Name}. Instead, you assign fonts to pre-defined categories: main sans mono alt drama deco.

This page explains how the font categories are used, and how to assign your choice of font to each category.

Novelette requires the included Swainson font, as main font. The included Towhee font will automatically be used for footnotes. You may not change these. Reason: It would take a lot of code branching, and waste a lot of time, to calculate layout for different main fonts. Swainson/Towhee are specifically designed for the intended purpose of Novelette. However, you may change whether or not certain OpenType features are used.

You may define several additional font categories. These are described below. But you may not create more categories/

OpenType features is on another page.

\mainfont{this command does not exist}

Just a reminder: You cannot choose the main font. If you attempt to do that, an error results.

\mainfeatures{OpenType features}

You may add (or remove) OpenType features from the main font. Normally, you do not need to do this, because the features you need (which may depend upon language setting) are selected automatically.

\sansfont{Font Family Name}

Traditionally, TeX defined a sans-serif font, primary for use in math expressions. But Novelette does not know, or care about, any math. You may set the sansfont to be a serif font, if you wish. The default is Swainson, same as main font.

\sansfeatures{OpenType Features}

Unless you command otherwise, the sans font will use the same features as the main font.

\monofont{Font Family Name}

Traditionally, TeX defined a monospace ("typewriter") font, for use in places where it was needed. But popular fiction has no need of monospace. You may cheat, and use any font. However, certain adjustments, such as kerning and character substitutions, are not automatically applied. The default font is Swainson, but if you decide to define the mono font, a better choice would be symbols or decorations, rather than flowing text.

\monofeatures{OpenType features}

Unless you command otherwise, no features are used, not even kerning or automated "TeX Ligatures."

\altfont{Font Family Name}

The alt font is intended as a "fallback font," when you need characters that are unavailable in Swainson. The substitution is not automatic; you must request the alt font where-needed. when-needed.

Font Libertinus Serif is strongly recommended as the alt font. It is available in TeX as package libertinus-fonts. This font plays well with Swainson, and has a large character set. If you do not set the alt font, Swainson will be used, but that defeats its purpose.

\altfeatures{OpenType Features}

Do what you need to do.

\dramafont{Font Family Name}

The defines a font that you may use for "dramatic effect," such as on the book title page. If you do not set it, Swainson is used.

\dramafeatures{OpenType Features}

Do what you need to do.

\decofont{Individual Font File}

The defines a font that you may use for decorations. Unlike the other font categories, this loads only a single font file, rather than the regular/italic/bold/bolditalic family.

Beware: Some decorative fonts have their characters traced from drawings. Often, such characters do not meet technical validation standards. Novelette cannot check this. They may appear good, but be flagged by the printer, because print-on-demand services use more stringent standards than needed by a home printer. On the other hand, some non-valid characters are acceptable.

Do what you need to do.

About Font Licenses

It is your responsibility to determine whether you are licensed to use fonts for your purpose. Presumably you intend commercial usage. Not all "free" fonts allow that.

Do not use GPL (GNU Public License) fonts, unless they come with an explicit font exception clause. Not all of them have it.

Some fonts are provided for use on platforms that have particular software installed, only when using that software.

The SIL Open Font License (OFL) is one of several licenses that are suitable. The LaTeX Project Public License (LPPL), and related GUST font license, are also good. There are others. If in doubt, read the license.
