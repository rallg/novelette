Fonts

Unlike most other LaTeX document classes, Novelette does not choose fonts via \usepackage{Font Name}. Instead, you assign fonts to pre-defined categories: main sans mono alt drama deco.

This page explains how the font categories are used, and how to assign your choice of font to each category.

The default main font is Swainson, which come with Novelette. This font is described below. You do not need to use it, but you might like it.

Other defaults are either the same as the main font, or one of the Latin Modern fonts. You may write an entire book without using any of these other categories. But if you do wish to use them, you will get better results if you assign your own choice of fonts.

Additional information regarding OpenType features is on another page.

\mainfont{Font Family}[OpenType Features]

The "main font" is used in main body text, and is also used anywhere you do not locally select a different font.

This command has identical effect to the \setmainfont command, of the fontspec package. You may use either command.

  • The Font Family might be something such as "Adobe Garamond Pro" or "Libertinus Serif" or "Clara". It does not need to be a LaTeX font. You may use any OpenType or TrueType font, as long as it is licensed for commercial usage.
  • The default mainfont is "Swainson", which comes with Novelette. If you use Swainson, then the "Towhee" font will be used for footnotes.
  • Some fonts, such as "Latin Modern Roman", are not designed for the purpose of popular fiction. They do not offer the same flowing readability as fonts designed for the purpose.
  • The font family does not need Bold or BoldItalic. These are never used in the main text of popular fiction.
  • Some OpenType features are pre-selected: kern,liga,onum. That is, kerning, common ligatures, oldstyle numerals. You do not need to write these features.
  • When the font is used in some locations, such as page headers, openings, and display pages, the ligatures are not used. This is because such text is often loose (greater space) or enlarged.
  • The TeX-specific "TeX Ligatures" are also pre-selected.

\sansfont{Font Family}[OpenType Features]

This command has identical effect to the \setsansfont command, of the fontspec package. You may use either command.

  • The default sansfont is Latin Modern Sans, which comes with LaTeX.
  • In principle, the sansfont should be be one of the sans-serif fonts. In practice, you may set any font, including those that have serifs.
  • The sansfont might be useful in special locations, such as chapter openings, and display pages.
  • In the document: \sans{This text uses sansfont}
  • Also see the above information for mainfont.

\monofont{Font Family}[OpenType Features]

This command has identical effect to the \setmonofont command, of the fontspec package. You may use either command.

  • You probably will not need to use monospaced text.
  • The default monofont is Latin Modern Mono, which comes with LaTeX.
  • Unlike other fonts, the monofont default features do not include kerning, ligatures, or oldstyle numbers.

  • A monospaced font is normally used for mathematics and tables, which are outside the purpose of Novelette. But if you need a monospace font, you may use it for special purposes. Or, you may assign a decorative font.
  • Some monospace fonts are designed to look like traditional typing, from a mechanical typewriters. Others have a more modern design, and are used by computer programmers.
  • In the document: \mono{This text uses monofont}
  • Also see the above information for mainfont.

\altfont{Font Family}[OpenType Features]

This command is specific to Novelette. It sets an "alternate font" that you can locally use, when you need characters that are not in the main font.

  • The default is Latin Modern Roman, which comes with LaTeX. But you should select your own alternate font.
  • A good general-purpose altfont is "Libertinus Serif". This font has a large character set, and a design style that is inconspicious (usually) when mingled with a different main font.
  • You can install Libertinus Serif via your TeX package manager. It is package "libertinus-fonts" (not any other package with similar name). You will still need to set \altfont.
  • Use of the altfont is not automatic. You must manually request it where you need it: \alt{This text uses altfont}
  • Also see the above information for mainfont.

\dramafont{Font Family}[OpenType Features]

This command is specific to Novelette. It sets a "dramatic font" that you can locally use, for special effect. That might be on the book title page, or in chapter openings.

  • You do not necessarily need a dramatic font.
  • The default dramafont is same as mainfont.
  • To use it: \drama{This is dramatic text}
  • Also see the above information for mainfont.

\decofont{font-filename.otf}[OpenType Features]

This command is specific to Novelette. It sets a "decorative font" that you can locally use, for dingbats or other decorative elements.

Unlike the other font settings, decofont does not select a font family. It selects a single font file, which you choose by its full file name, with extension.

  • You do not necessarily need a decofont.
  • The default decofont is Swainson, same as mainfont. Note that Swainson does not have any decorative characters, so this is not helpful.
  • The decofont default features do not include kerning, ligatures, or oldstyle numbers.
  • To use it: \deco{character}
  • Novelette does not support drop caps (multi-line first letter).
  • Beware: Some fonts have characters traced from drawings. These fonts may fail validation tests at the manufacturer (Novelette will not complain). If you are using such characters, it is better to convert them to raster images.

About Font Licenses

It is your responsibility to determine whether you are licensed to use fonts for your purpose. Presumably you intend commercial usage. Not all "free" fonts allow that.

Do not use GPL (GNU Public License) fonts, unless they come with an explicit font exception clause. Not all of them have it.

Some fonts are provided for use on platforms that have particular software installed, only when using that software.

The SIL Open Font License (OFL) is one of several licenses that are suitable. The LaTeX Project Public License (LPPL), and related GUST font license, are also good. There are others. If in doubt, read the license.

About Swainson and Towhee Fonts

These OFL-licensed fonts were designed by the Novelette developer, using FontForge software. The OFL-licensed Cochineal fonts were used as reference, but the glyph shapes are different.

  • Swainson is not a clone or revival or any other font. However, it has the general readability of fonts such as Garamond, Bembo, and Caslon.
  • The character set includes Latin-1, plus a variety of punctuation and typographic support, and a few letters outside Latin-1. Modern Western European languages are supported, including Irish Gaelic and Welsh. Some languages beyond Western Europe are also supported, such as Finnish, Swahili, Afrikaans, Hawai'an, and Filipino.
  • Swainson does not support Eastern European languages. Novelette will support languages such as Czech or Polish, but then you will need to use a different font.
  • Whether or not you use Swainson, Novelette does not support right-to-left languages, and does not support languages with stacked (multiple) diacritical marks.
  • If you use Swainson as main font, then the related Towhee font will be automatically used for small text (footnotes). Towhee is a modified form of Swainson, intended for this purpose.
